# bluelibrary

An extended collection of high-performance **Bluespec SystemVerilog (BSV)** and **Verilog** libraries tailored for modern AMD FPGA architectures (e.g., Xilinx UltraScale+, Alveo). 

## About This Repository

The standard Bluespec (BSV) library is powerful, but it often lacks direct support for cutting-edge hardware features specific to certain FPGA vendors (such as Xilinx's UltraRAM). **blueLibrary** was created to fill this gap. 

This repository serves as a personal and public archive for custom hardware modules, wrappers, and interfaces that provide **100% Drop-in Replacement** compatibility with standard BSV libraries, while unlocking the full potential of the underlying silicon.
